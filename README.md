# Previsão de Diabetes em Endpoint de Modelo criado no Databricks

Esse é um projeto pessoal a qual irei criar um ambiente governado no databricks com acesso a colegas com papeis de: analista, governancia e cientista de dados, o foco, além da analise dos dados e toda a pipeline de treinamento do modelo, será realizar o tracking do modelo com mlflow, deploy no UC e servi-lo com endpoint pronto para consumo.
